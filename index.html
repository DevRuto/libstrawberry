libstrawberry

View Repository

libstrawberry is a young, open-source, cross-platform cryptography and networking library written in C, licensed under the MIT license. It aims to provide a feature-rich, consistent, simplified programming interface. Neither strawberries nor cupcakes included.

All functions are prefixed with the sb_ prefix.

Features

  • Core
    • Time
      • Acquiring timestamps and the TSC
      • Sleeping
    • Memory
      Underlined functions have a _s (memory locking) and a _u (normal) variant.
      • malloc: memory allocation
      • calloc: cleared memory allocation
      • realloc: memory reallocation
      • cpyalloc: cloning memory allocation
      • ntcpyalloc: null-terminated cloning memory allocation
      • free: frees memory, always returns NULL
      • memset: sets memory
      • memcpy: copies memory
      • memcmp: compares memory
      • memequ: checks if memory is equal
      • memdump: dumps the specified region in the memory to stdout
      • strcpy: copies a string
      • strappend: appends to a string
    • Error Reporting
      Library functions set error codes and error parameters if things go south, which you can use to diagnose what's going wrong.
      • error_get
      • error_get_param
      • error_print
  • Cryptography
    • Helpers
      • Generalized encryption/decryption: easy symmetric encryption and decryption of data using one interface.
      • Key generation: create keys based on an input seed.
      • Global PRNG (ISAAC): acquire random 16-bit, 32-bit, and 64-bit numbers without a hassle.
    • Hashing
      • MD5
      • RIPEMD160
      • SHA256
      • SHA512
    • Key Exchange
      • Diffie Hellman
    • Padding
      • PKCS#7
      • ISO 9797-1 #1 (zero padding)
      • ISO 9797-1 #2
    • Pseudo-random Number Generators
      • ISAAC
    • Symmetric
      • Rabbit
      • Rijndael
      • Salsa20
  • Miscellaneous
    • GMP
      • Integer import
      • Integer export
  • Networking
    • Simplified sockets.
      • socket_init
      • socket_clear
      • socket_start
      • socket_stop
      • socket_accept
      • socket_acceptfd
      • socket_read
      • socket_write
