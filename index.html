libstrawberry

View Repository

libstrawberry is a young, cross-platform cryptography and networking library written in C.
It aims to provide a feature-rich, consistent, simplified programming interface.
Neither strawberries nor cupcakes included.


Features

  • Core
    • Time
      Acquire timestamps, sleep.
    • Memory
      Memory helpers, safe (locks memory) and unsafe variants of functions such as malloc, calloc, realloc, cpyalloc, and others.
    • Error Reporting
      Library functions set error codes and error parameters if things go south, you can use these to diagnose what's going wrong.
  • Cryptography
    • Helpers
      Generalized encryption/decryption: easy symmetric encryption and decryption of data using one interface.
      Key generation: create keys based on an input seed.
      Global PRNG (ISAAC): acquire random 16-bit, 32-bit, and 64-bit numbers without a hassle.
    • Hashing
      • MD5
      • RIPEMD160
      • SHA256
      • SHA512
    • Key Exchange
      • Diffie Hellman
    • Padding
      • PKCS#7
      • ISO 9797-1 #1 (zero padding)
      • ISO 9797-1 #2
    • Pseudo-random Number Generators
      • ISAAC
    • Symmetric
      • Rabbit
      • Rijndael
      • Salsa20
  • Miscellaneous
    • GMP
      • Integer import
      • Integer export
  • Networking
    • Simplified sockets.
Powered by a very beautiful cupcake.
